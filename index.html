I selected the command line section as a way to learn more about computing.
 The command line as a whole can be slighlty intimidating but it has a lot of 
 potential that people don't realize. Here's what I learned...

 • The command line is a text interface that 
 communicates directly with your computer's Operating System (OS)

 • You can use it to display your computer id, print 
 out statements, an other basic actions.

 • A basic but useful action is navigating the directory 
of your computer through commands like "ls" which 
gives lists of all direcotry files. 

• If you'd ever like to change your directory or where 
you are in your computer you can use "cd /" or "cd~/" 
in order to return home

• Finally a useful and time saving use for the command line 
is the creating, writing and running of files***. 

    *** You can create a document, add content to it (for 
    instance a snippet of code written in python) and the 
    command line will write (create and execute the file)
    and return the result of whatever script is created.***

The Command Line can be used to manipulate files on your computer's OS in a
